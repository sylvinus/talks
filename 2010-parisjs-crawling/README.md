# Web Crawling with NodeJS

- **Event:** ParisJS #2
- **Date:** November 25, 2010
- **Location:** Paris, France
- **Format:** Lightning talk

## Description

An early talk on using Node.js for web crawling, presented at the second ParisJS meetup. This talk kickstarted the [node-crawler](https://github.com/sylvinus/node-crawler) project, covering HTML parsing with Apricot and jsdom, concurrency management with generic-pool, and a simple crawler API.
