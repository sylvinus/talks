# Developer-friendly task queues: what we learned building MRQ

- **Event:** [PyParis 2017](https://2017.pyparis.org/)
- **Date:** June 12, 2017
- **Location:** Pôle Universitaire Léonard de Vinci, La Défense, Paris, France

## Description

Lessons learned building [MRQ](https://github.com/pricingassistant/mrq), a distributed task queue for Python built on top of MongoDB and Redis. MRQ was developed at Pricing Assistant to replace Celery/RQ with a developer-friendly alternative focused on monitoring, debugging, and ease of use for data-intensive workloads.
